# 急症识别与风险分层体系升级总结

## 📅 升级日期
2026年5月10日

## 🎯 升级目标

将系统从"关键词演示版"升级为**具备医疗初筛产品思维的 Red Flag Triage MVP**，解决严重的风险分层缺陷。

## ❌ 原系统问题

### 关键Bug
输入"被车撞了 大出血"系统返回：
- ❌ 风险等级：LOW
- ❌ 风险指数：0/100
- ❌ 建议："可观察，注意变化"

这是明显错误！这类输入包含明显的外伤急症和出血风险，不应判定为低风险。

### 根本原因
- 缺少外伤/事故相关规则
- 没有系统化的急症识别体系
- 只有零散关键词匹配，无法覆盖自然语言表达
- 风险评分机制不合理
- 建议生成逻辑与风险等级不一致

---

## ✅ 升级内容

### 1️⃣ 重构风险引擎核心规则

#### 建立完整的 RED FLAG 体系

新增 **EMERGENCY** 等级，现在支持4个风险等级：
- **EMERGENCY** - 需要立即急救/拨打120
- **HIGH** - 高危风险，需尽快就医
- **MEDIUM** - 中等风险，建议24-48小时内就诊
- **LOW** - 低风险，可观察

#### 覆盖 9 大高危类别

##### 【外伤/事故/暴力损伤】
- 车祸、交通事故、被车撞
- 高处坠落、摔伤、头部受伤
- 刀伤、砍伤、暴力伤害
- 骨折、骨头断裂、肢体变形

##### 【出血/失血风险】
- 大出血、流血不止、止不住血
- 呕血、吐血、黑便
- 咯血不止、大量咯血
- 产后大出血、孕期大出血

##### 【呼吸循环系统危急】
- 喘不上气、呼吸困难、窒息
- 胸痛压榨感、疑似心梗
- 晕倒、晕厥、休克

##### 【神经系统急症】
- 意识不清、昏迷、叫不醒
- 中风征兆（单侧无力、口眼歪斜、说话不清）
- 霹雳样剧烈头痛
- 抽搐、痉挛

##### 【急腹症】
- 剧烈腹痛、腹痛难忍
- 右下腹痛（疑似阑尾炎）
- 腹膜刺激征（按压更痛、腹部板状）

##### 【高热/感染重症】
- 高热（≥39℃）
- 脑膜炎征象（颈部僵硬+高热）

##### 【严重过敏/中毒】
- 过敏性休克、喉头水肿
- 药物中毒、农药中毒

##### 【精神心理危机】
- 自杀意念、自残倾向
- "不想活了"、"想死"

##### 【特殊人群风险】
- 孕产妇危险征象
- 儿童高危症状
- 老年人特殊风险

### 2️⃣ 优化风险评分机制

#### 新评分规则
- **EMERGENCY**: 90-100分
- **HIGH**: 70-90分
- **MEDIUM**: 40-70分
- **LOW**: 10-40分

#### 智能加分机制
1. 多规则叠加：每个额外规则 +2分（上限+10）
2. 跨类别组合：症状跨类别 +5分
3. 不再出现"明显高危输入却0分"的情况

#### 示例对比

| 输入 | 旧系统 | 新系统 |
|------|--------|--------|
| "被车撞了 大出血" | LOW, 0分 | ✅ EMERGENCY, 99分 |
| "胸痛喘不上气" | 不确定 | ✅ HIGH, 89分 |
| "头痛三天，发烧38.5度" | 可能MEDIUM | ✅ MEDIUM, 64分 |

### 3️⃣ 增强自然语言理解

#### 支持口语化表达
- ❌ 旧：只匹配标准医学词汇
- ✅ 新：支持"被车撞了"、"流血不止"、"快不行了"

#### 识别强度修饰词
- "很"、"非常"、"特别"、"极其"、"剧烈"、"严重"
- "止不住"、"停不下"、"一直"、"不停"

#### 识别趋势词
- "越来越"、"更加"、"加重"、"恶化"、"变重"

#### 识别时间因素
- 持续时间：3天以上、一直、好几天
- 突然性：突然、突发

### 4️⃣ 完善建议生成逻辑

#### EMERGENCY 级别建议
```
⚠️ 紧急情况！请立即采取行动：

1. 立即拨打急救电话 120 或 999
2. 或立即前往最近医院急诊室
3. 在等待急救时，保持镇静，寻求身边人帮助
4. 如有生命危险征象，请立即现场急救

```

```
您的症状提示存在较高健康风险，建议：

• 尽快前往医院急诊或专科门诊就诊
• 建议在今日内完成就医评估
• 不要自行用药，避免掩盖病情
• 如症状突然加重，请直接前往急诊
```

#### MEDIUM & LOW 建议
- MEDIUM：24-48小时内就诊
- LOW：可观察1-2天，症状加重及时就医

#### 一致性保证
- ❌ 旧：高危输入却建议"先观察"
- ✅ 新：建议完全匹配风险等级，不会出现矛盾

### 7️⃣ 强化 Fallback 安全兜底

#### Rules-Only 模式保障

即使没有 `OPENAI_API_KEY` 或 LLM 调用失败，规则引擎仍能：
- ✅ 正确识别外伤、出血、胸痛、呼吸困难
- ✅ 正确识别意识异常、自杀风险
- ✅ 输出安全的标准建议
- ✅ 绝不降低风险等级

#### 三层安全机制
```
Layer 1: LLM + 规则引擎（最佳）
         ↓ LLM失败
Layer 2: 规则引擎 + 关键词提取（安全）
         ↓ 规则失败
Layer 3: 保守策略（MEDIUM + 建议就医）
```

```python
# 智能评分
def _calculate_risk_score(risk_level, num_hits, has_multiple_categories):
    base_scores = {"EMERGENCY": 95, "HIGH": 80, "MEDIUM": 55, "LOW": 20}
    score = base_scores[risk_level]
    if num_hits > 1: score += min((num_hits - 1) * 2, 10)
    if has_multiple_categories: score += 5
    return min(score, 100)
```
